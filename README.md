# DS-Final-Project---Logan-Amaya-Daylon
This project explores whether Olympic athletes naturally group into distinct archetypes based on physical characteristics and sport participation using unsupervised machine learning techniques. Using the “120 Years of Olympic History” dataset from Kaggle, we analyzed athlete features such as age, height, weight, sex, and sport to uncover hidden patterns across Olympic competitors. To prepare the data, we removed missing values, standardized numerical features, encoded categorical variables, and applied dimensionality to improve clustering performance. We then implemented and compared Agglomerative Clustering, K-Means, and DBSCAN to identify meaningful athlete groupings.
